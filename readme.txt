== Description ==

This plugin create calendar shortcode of 1 month Unit.
You can relate tagged post to date. Tag has been set by List Calendar > Settings.

== Installation ==

1. Upload `list-calendar` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to displays tagged post to List Calendar > List > Add New =

First, you must make the following settings.

1. Set tag on List Calendar > Setting.
1. Put above tag to post. 

Click List Calendar > List > Add New, you can relate post tagged to date.
